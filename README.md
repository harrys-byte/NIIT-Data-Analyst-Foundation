This repository contains the practical tasks, exercises, and mini-projects completed as part of the **NIIT Data Analyst Foundation** program. The work here reflects hands-on learning across core data analysis concepts and tools.

## Program Coverage

The tasks in this repository span the following key areas:

- **Data Analysis Fundamentals**
  - Understanding data types, data cleaning, and exploratory analysis
- **Microsoft Excel**
  - Data cleaning, formulas, pivot tables, and basic analysis
- **Python for Data Analysis**
  - Data manipulation using Pandas and NumPy
  - Basic scripting and logic building
- **Statistics & Mathematics**
  - Descriptive statistics, probability, and basic inferential concepts
- **Data Visualization**
  - Creating charts and visual insights using Python and Excel
- **Mini Case Studies / Assignments**
  - Applying analytical thinking to solve real-world data problems
